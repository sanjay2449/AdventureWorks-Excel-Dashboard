# 🚴 AdventureWorks Executive Sales Dashboard | Excel Data Analytics Project

![Dashboard Preview](Dashboard_Screenshot.png)

## 📌 Project Overview

This project demonstrates an end-to-end Excel Data Analytics solution built using the AdventureWorks dataset. The objective was to transform raw sales, customer, product, and territory data into an interactive executive dashboard that helps stakeholders monitor business performance and make data-driven decisions.

The project covers the complete analytics workflow including data cleaning, data modeling, DAX calculations, KPI reporting, and dashboard development.

---

# 🎯 Objectives

- Clean and transform raw data using Power Query
- Build a scalable Data Model using Power Pivot
- Create relationships between fact and dimension tables
- Write DAX measures for KPIs
- Build an interactive Executive Dashboard
- Generate meaningful business insights

# 🧹 Data Cleaning (Power Query)

The following data preparation steps were performed:

- Removed unnecessary columns
- Corrected data types
- Replaced missing values
- Cleaned Product Color values
- Categorized Product Size into Size Groups
- Created custom columns
- Created conditional columns
- Merged required tables
- Appended yearly sales data
- Loaded cleaned data into the Excel Data Model

---

# 🗂 Data Model

A Star Schema data model was created using:

### Fact Table

- All_Sales_Data

### Dimension Tables

- Customer
- Product
- Product Subcategory
- Product Category
- Territory Lookup
- Calendar
- Returns Data

Relationships were created using primary and foreign keys for efficient filtering and analysis.

---

# 📊 DAX Measures

The following DAX measures were created:

- Total Revenue
- Total Cost
- Total Profit
- Total Quantity
- Total Transactions
- Profit Margin %

These measures dynamically update based on slicer selections.
